# Fibonacci sequence app
Multi container\
Calculate fibonacci number by redis based on input index.\
Store inputs in posgreSQL database.

## Technologies stack
- CRA
- Express
- Redis
- Postgres
- Docker
- Docker-compose
- Travis CI
- Elastic Beanstalk

## Commands
- docker-compose up --build
- docker-compose down
